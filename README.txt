Voilà le premier projet de l'année en Technologie du Web.
Son principe est simple, coder un site en utilisant dans un premier temps des page xml, les pages xml sont alors converties en xhtml avec une page xls.
Avec les pages xml une DTD est évidement crée.
Puis ensuite la création de quelques pages xhtml avec leur style en CSS.
